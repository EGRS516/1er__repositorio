# Pagina sencilla

Pequeña plantilla estática en español: responsive, accesible y lista para desplegar.

## ¿Qué es?
una página estática de ejemplo que incluye estructura HTML, CSS y JavaScript minimalista para interacciones básicas (menú móvil, formulario simulado, botón de color aleatorio).

## Uso
- Abrir `index.html` directamente en el navegador.
- Para servir en un servidor local (recomendado), desde la carpeta del proyecto ejecutar:

  `python -m http.server 8000`

  y abrir `http://localhost:8000`.

## Estructura principal
- `index.html` — Página principal.
- `css/styles.css` — Estilos.
- `js/app.js` — Interacciones JavaScript.
